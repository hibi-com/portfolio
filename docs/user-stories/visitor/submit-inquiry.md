title: "問い合わせを送信する"
---

## ユーザーストーリー

**As a** サイト訪問者
**I want to** 問い合わせフォームから連絡したい
**So that** 質問や依頼を送信できる

## 受け入れ基準

1. 問い合わせフォームにアクセスできる
2. 件名、内容、メールアドレスを入力できる
3. バリデーションエラーが適切に表示される
4. 送信成功時に完了メッセージが表示される

## シナリオ

### シナリオ1: 問い合わせを正常に送信する

**Given:** お問い合わせページにアクセスしている
**When:** 件名「技術的な質問」を入力する
**And:** 内容「この機能について質問があります」を入力する
**And:** メールアドレス「user@example.com」を入力する
**And:** 送信ボタンをクリックする
**Then:** 送信完了メッセージが表示される

### シナリオ2: 必須項目が未入力の場合

**Given:** お問い合わせページにアクセスしている
**When:** 件名を空のまま送信ボタンをクリックする
**Then:** 「件名は必須です」エラーメッセージが表示される

### シナリオ3: メールアドレスが不正な形式の場合

**Given:** お問い合わせページにアクセスしている
**When:** メールアドレスに「invalid-email」を入力する
**And:** 送信ボタンをクリックする
**Then:** 「メールアドレスの形式が正しくありません」エラーメッセージが表示される
