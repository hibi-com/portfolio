title: "ポートフォリオを閲覧する"
---

## ユーザーストーリー

**As a** サイト訪問者
**I want to** ポートフォリオを閲覧したい
**So that** 制作実績やスキルを確認できる

## 受け入れ基準

1. ポートフォリオ一覧ページにアクセスできる
2. プロジェクトのタイトル、会社名、サムネイルが表示される
3. プロジェクトをクリックすると詳細ページに遷移する
4. 詳細ページでプロジェクトの説明、技術スタック、画像が表示される

## シナリオ

### シナリオ1: ポートフォリオ一覧からポートフォリオ詳細へ遷移する

**Given:** トップページにアクセスしている
**When:** ポートフォリオ一覧リンクをクリックする
**Then:** ポートフォリオ一覧ページが表示される

**When:** 最初のポートフォリオカードをクリックする
**Then:** ポートフォリオ詳細ページが表示される
**And:** プロジェクトのタイトル、説明、技術スタックが表示される

### シナリオ2: 存在しないポートフォリオにアクセスした場合

**Given:** 存在しないポートフォリオのURLにアクセスする
**When:** ページが読み込まれる
**Then:** 404エラーページが表示される
